# 前端学习与刷题系统

面向中高级前端学习、作业练习、错题复盘和知识文档沉淀的个人学习系统。

线上地址：[https://aoyia.github.io/frontend-learning-system/](https://aoyia.github.io/frontend-learning-system/)

## 本地启动

```bash
cd 学习系统
npm install
npm run dev
```

## 构建

```bash
cd 学习系统
npm run build
```

## 内容维护

- `技术破冰/`：知识图谱和破冰导读
- `学习系统/docs/博客文档/`：博客、笔记和专题 Markdown 文档
- `学习系统/data/`：学习系统数据与长文内容
