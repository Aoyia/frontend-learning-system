# 性能监控与性能预算

## 1. 它属于哪个知识板块？

```txt
前端性能优化
→ 性能治理
→ RUM / 性能预算 / CI 门禁 / 发布回归
```

==性能监控与预算解决的是“优化完会不会又退化”的问题。==

## 2. 陌生技术出现后的学习顺序

### 2.1 它解决什么问题？

==**性能监控与性能预算**解决的是“**性能优化只做了一次，后面每次发布都可能慢回去**”的问题。==

性能退化通常不是一次大事故，而是每次加一点依赖、加一点脚本、加一点接口、加一点图片，最后突然发现页面已经很慢。

==性能治理的核心是把性能变成**持续可见的指标**和**发布前后的约束**，而不是靠某次专项优化。==

### 2.2 核心流程

```txt
确定关键页面和关键路径
→ 定义预算：LCP / INP / CLS / JS 体积 / 图片体积 / 请求数
→ CI 中检查构建产物和 Lighthouse
→ 线上采集 RUM
→ 发布后按版本对比
→ 超过阈值报警或阻断
→ 复盘退化来源并回滚或修复
```

### 2.3 关键词清单

1. RUM：Real User Monitoring，真实用户性能监控。
2. performance budget：性能预算，提前定义不能超过的指标阈值。
3. regression：性能回归，某次改动导致指标变差。
4. release dimension：按版本号、commit、灰度批次分析性能。
5. page dimension：按页面、路由、业务入口分析性能。
6. device dimension：按设备、浏览器、网络分组。
7. synthetic monitoring：定时跑的合成监控，环境可控。
8. CI gate：CI 门禁，例如 bundle size 超限就失败。
9. alerting：指标超过阈值后通知。
10. rollback signal：性能指标作为回滚或停止放量依据。

### 2.4 一句面试版

==性能监控与预算的核心是把 **LCP、INP、CLS、资源体积、请求数** 等指标纳入持续治理：CI 阶段检查构建产物和预算，线上通过 **RUM** 按页面、设备、网络和版本采集真实数据，发现性能回归后能报警、定位和回滚。==

### 2.5 最小 demo / 最小案例

最小预算可以先写成表：

```txt
首页：
- LCP p75 <= 2.5s
- INP p75 <= 200ms
- CLS p75 <= 0.1
- 首屏 JS gzip <= 180KB
- 首屏图片 <= 300KB

后台列表页：
- INP p75 <= 200ms
- 首次表格可交互 <= 3s
- 一次筛选不产生超过 100ms 主线程任务
```

构建阶段检查资源体积：

```txt
build
→ 读取 dist/assets
→ 统计 JS / CSS / image gzip 体积
→ 超过预算直接失败或给出警告
```

线上阶段检查真实体验：

```txt
RUM 上报
→ page + version + device + network + metric
→ 看 p75 / p95
→ 与上一版本对比
```

### 2.6 常见治理动作

1. 每个关键页面定义 3-5 个核心指标，不要什么都管。
2. 每次构建记录 JS/CSS/图片体积变化。
3. 每次发布带上版本号，方便按版本对比。
4. RUM 指标必须带页面、设备、网络、浏览器、版本。
5. 灰度发布时把 LCP、INP、错误率、接口耗时一起看。
6. 性能回归要能定位到版本、页面、资源或交互。
7. 预算不是一次定死，要随业务和设备分层调整。

### 2.7 和 Chrome Memory 的关系

[[Chrome Memory 与 Performance Monitor|Chrome Memory / Performance Monitor]] 更偏本地诊断；性能监控与预算更偏线上治理。前者帮你定位“为什么慢”，后者帮你发现“什么时候又慢了”。

## 3. 选择题自测

### Q1

性能预算的作用是什么？

A. 在发布前后约束性能指标，防止持续退化
B. 阻断自动化需求变更以代替人工需求评审
C. 强制在构建时把全部图片和多媒体资源物理剔除
D. 让所有的异步网络请求均在服务端完成同构拦截

答案：A

### Q2

RUM 指标为什么要带版本号？

A. 方便判断哪次发布导致性能回归
B. 方便改标题
C. 方便删除日志
D. 方便隐藏错误

答案：A

### Q3

CI 阶段适合检查什么？

A. 构建产物体积和预算是否超限
B. 用户是否喜欢设计稿
C. 线上真实网络是否稳定
D. 用户电脑是否太旧

答案：A
